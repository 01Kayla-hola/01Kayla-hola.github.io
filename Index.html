Computer Hardware

A computer consists of many different hardwares. Below are the common hardwares in a computer:

1. CPU (Central Processing Unit) or Processor

CPU is used to interpret and execute commands in a computer by performing aritmetical calculation.

1.2 Intel CPU

Intel CPU has been around since 1968. Some people use Intel because of their preference, but now the performance is comparable with AMD.

1.2 AMD CPU

AMD entered CPU market in 1975. Before, the quality of AMD CPU was below Intel, but recently it has become better and even surpassed Intel in some aspect.

2. RAM (Random Acess Memory)

The more RAM you have, the more apps you can run at the same time. The data inside RAM will be deleted when the app closes or the computer shut down.

3. HDD (Hard Disk Drive)

The build of hard disk actually makes it unfriendly for mobile use, since the disk is sensitive to physical shock when being used, such as rotating or moving it too roughly.
